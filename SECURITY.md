# Security policy for {{ORG_DISPLAY_NAME}}

## Reporting a vulnerability

If you discover a security vulnerability in any repo under `{{ORG_SLUG}}`, email **security@simplemotion.com** with:

- Repository and commit SHA (or version)
- Steps to reproduce
- Impact assessment

**Do not open a public issue** for security reports — doing so before a fix ships risks exposing users. We aim to acknowledge reports within one business day.

## Supported versions

`main` is the supported branch. Historical releases receive security fixes on a case-by-case basis.
