# Contributing to {{ORG_DISPLAY_NAME}}

Thanks for contributing. These defaults apply to every repo under `{{ORG_SLUG}}` unless overridden per-repo.

## Before you open a PR

- Check existing issues and open PRs to avoid duplication.
- Keep PRs small and focused on one concern.
- Reference the issue you're addressing; explain the "why", not just the "what".

## Commit and PR hygiene

- Write one-line summaries at the start of commit messages.
- Squash-merge unless there's a concrete reason to preserve individual commits.
- Rebase on top of the latest `main` before requesting review.

## IP assignment

Contributions are assigned to SimpleMotion.Global Pty Ltd per `ASSIGN.md` in every SimpleMotion repo.
